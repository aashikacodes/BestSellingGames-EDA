# Best Selling Video Games – EDA Project

## Overview
This project performs Exploratory Data Analysis (EDA) on a dataset of best-selling video games using Pandas and Matplotlib.

## Dataset
Source: Kaggle  
Contains information about game titles, platforms, publishers, and global sales.

## Tools Used
- Python
- Pandas
- Matplotlib

## Analysis Performed
- Data cleaning and missing value handling
- Top selling games analysis
- Publisher-wise and platform-wise sales comparison
- Visual insights using bar charts

## Key Insights
- Nintendo dominates the list of best-selling games
- Multi-platform games tend to have higher total sales
